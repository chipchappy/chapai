# ChapAI SEO Discoverability Execution Sheet

## Goal
Increase qualified traffic for CCRN and NCLEX by shipping high-intent pages that match what buyers are already searching for.

## Scope
- CCRN topic pages
- NCLEX topic pages
- Comparison pages
- Study-plan pages
- No app code changes in this patch

## Rules
- One page = one primary intent.
- Put the keyword in the title, H1, first paragraph, and meta description.
- Every page must link to the right package page.
- Avoid thin pages that repeat the same copy with a different title.
- Keep each page short, specific, and useful.

## Execution Steps
1. Publish 2 CCRN pages and 2 NCLEX pages first.
2. Publish 2 comparison pages next.
3. Publish 2 study-plan pages next.
4. Interlink all pages back to `/upgrade`.
5. Review clicks and iterate only on pages that earn traffic.

## Recommended Page Template
- Clear promise in the hero
- Short explanation of who the page is for
- One proof block with original questions or rationale
- One CTA to the correct package
- One FAQ block for hesitation

## Success Metrics
- More indexed pages
- More search impressions
- More clicks to package pages
- More purchases from search-intent traffic
