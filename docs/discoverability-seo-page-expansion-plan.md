# ChapAI SEO + Page Expansion Plan

## Goal
Turn ChapAI into a discoverable product for two separate markets:
- CCRN ICU nurses
- NCLEX nursing students

## What to build next
1. Keep the homepage focused on the two package paths and one premium proof section.
2. Build high-intent pages for each exam topic cluster.
3. Build comparison pages for users already shopping.
4. Build study-plan pages for users who want structure, not just questions.
5. Reuse the same content template across pages so they ship quickly.

## Page template
Each page should include:
- one clear promise in the first screen
- one short explanation of who it is for
- one proof block with original question count or rationale sample
- one CTA to the correct package page
- one FAQ block targeting buyer hesitation

## SEO rules
- Use one primary keyword per page.
- Put the keyword in the title, H1, meta description, and first paragraph.
- Keep internal links pointing to the right exam package.
- Avoid thin pages that say the same thing with a different title.

## Fastest path to traffic
- Ship 2 CCRN pages and 2 NCLEX pages first.
- Ship 2 comparison pages next.
- Add 1 study-plan page per exam.
- Link all of them to `/upgrade`.

## Success metric
- More indexed pages.
- More qualified clicks.
- More package-page visits.
- More purchases from search intent.
