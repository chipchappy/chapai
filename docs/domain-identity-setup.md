## Domain identities

Recommended aliases for `chapaisolutions.com`:

- `hello@chapaisolutions.com`
- `social@chapaisolutions.com`
- `creators@chapaisolutions.com`
- `daily@chapaisolutions.com`
- `partnerships@chapaisolutions.com`

Recommended first live configuration:

- `CHAPAI_EMAIL_PROVIDER=resend`
- `CHAPAI_EMAIL_FROM=daily@chapaisolutions.com`
- `CHAPAI_EMAIL_API_KEY=<provider key>`

Recommended social approval identity:

- use `social@chapaisolutions.com` in creator outreach signatures
- use `creators@chapaisolutions.com` for formal creator partnership replies
- use `partnerships@chapaisolutions.com` for educator / hospital system outreach

These identities are not provisioned automatically by the app. They still require mailbox or provider setup on the domain host.
