# Daily Question Email Segmentation Plan

## Segments
- `CCRN only`
- `NCLEX only`
- `Both / undecided`
- `Educators`

## Send logic
- CCRN leads get critical-care heavy questions first
- NCLEX leads get priority / delegation / safety first
- Both leads get the cleanest “best of” question and a package split CTA
- Educators get a learner-facing question plus a pilot CTA every 3-4 sends

## First pilot goals
- prove opens
- prove clicks
- learn which exam segment converts fastest
- build a habit loop before automation complexity grows

## First metrics to watch
- open rate
- click-through rate
- reply rate
- plan page visits
- sprint-pass conversions
