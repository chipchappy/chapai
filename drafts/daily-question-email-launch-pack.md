# Daily Question Email Launch Pack

## Goal
Turn cold visitors and social traffic into repeat readers with one clean daily question email.

## Core promise
- one question
- one teaching point
- one link back into ChapAI
- one soft CTA to the `$10` cram pass

## Daily structure
1. short subject line
2. one question stem
3. four options or a simplified bedside prompt
4. one-click answer reveal link
5. one-sentence rationale teaser
6. CTA: `Open today’s full rationale`

## Segments
- CCRN: hemodynamics, vents, shock, drips, neuro
- NCLEX: delegation, safety, pharm, prioritization
- educator / institution: curriculum-friendly weekly digest

## First send rhythm
- weekday only
- 6:30am local send for working nurses
- one weekend recap email on Sunday

## CTA ladder
- free question
- `$10` cram pass
- Plus / Pro
