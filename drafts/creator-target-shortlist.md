# Creator Target Shortlist

Use this as the ranked working sheet for the first outreach wave. Keep it small, segmented, and practical.

## How to use it
- build `10` CCRN / ICU targets
- build `10` NCLEX / nursing-student targets
- rank by trust and likely response, not vanity reach
- pick a best hook, best package, and likely price band before sending anything

## Fields
- name
- platform
- segment
- audience fit score
- trust score
- likely price band
- best hook
- best package
- first outreach channel
- next move
