# Creator Offer Sheet

## Simple founder-collab offer

### Base offer
- free founder pass
- unique code or link
- one simple deliverable
- natural tone, not scripted ad copy

### Nano creator band
- audience: small but trusted
- budget: `$75-$150`
- deliverable: `1` story set or `1` short reel
- best for: first testing, warm educator pages, study-with-me pages

### Micro creator band
- audience: stronger reach and more reliable conversion volume
- budget: `$150-$300`
- deliverable: `1` reel or `1` short honest review thread/story bundle
- best for: the first real repeatable conversion test

### Optional upside
- custom code
- simple commission if they want it
- no exclusivity on first test

## What to ask for
- natural creator voice
- honest first impression
- one clean CTA
- link to the right package page, not a random homepage
